# Implementační plán — Milník 2 (KPI core)

**Datum:** 2026-04-24 · **Navazuje na:** M0 + M1 (plán `2026-04-24-m0-m1-foundation-and-data.md`)

**Cíl M2:** Postavit KPI katalog všech 20 KPI a deterministickou analytickou vrstvu (KPIEvaluator, DriverAnalyzer, AnomalyDetector, NarrativeGenerator, ActionRecommender, AIInsightProvider mock) + formula calculators pro Priority 1 KPIs. Výsledkem je schopnost získat z `MockDataProvider` kompletní `KPIEvaluation` objekt, který UI bude konzumovat v M3.

**Architektura:**

```
DataProvider → KPICalculator (per KPI) → KPIEvaluator → {KPIEvaluation}
                                             ↓
                         DriverAnalyzer · AnomalyDetector
                                             ↓
                         NarrativeGenerator · ActionRecommender · AIInsightProvider
                                             ↓
                                   {EnrichedKPIEvaluation}
```

## Rozsah

### In scope M2
1. `lib/kpi/types.ts` — KPIDefinition, SectionId, DataSource, KPIEvaluation, KPIDriver, KPIEnriched
2. `lib/kpi/catalog.ts` — definice všech 20 KPI (thresholds, owner, frequency, action, …)
3. `lib/analytics/period.ts` — helpery pro month/quarter/half-year ranges
4. `lib/analytics/kpi-evaluator.ts` — KPIEvaluator (status, trend, sparkline)
5. `lib/analytics/driver-analyzer.ts` — DriverAnalyzer (segment ranking)
6. `lib/analytics/anomaly-detector.ts` — AnomalyDetector (z-score nad sparkline)
7. `lib/analytics/narrative-generator.ts` — NarrativeGenerator (CZ šablony)
8. `lib/analytics/action-recommender.ts` — ActionRecommender
9. `lib/analytics/ai-insight-provider.ts` — `AIInsightProvider` interface + `MockAIInsightProvider`
10. `content/ai-insights/mock-insights.json` — pre-written AI insights per KPI
11. `lib/kpi/calculators/` — výpočty hodnot a sparklines pro P1 KPIs:
    - `fluct.ts` — fluktuace overall (odchody / avg HC za období)
    - `fluct-crit.ts` — fluktuace kritických pozic
    - `hc-fte-div.ts` — HC/FTE per divize (vacancy rate vs. budget)
    - `wf-movement.ts` — nástupy/odchody per měsíc/divizi
    - `avg-wage.ts` — průměrná mzda per FTE
    - `holiday-untaken.ts` — zůstatek nevyčerpané dovolené (mock)
    - `hr-stats.ts` — agregované HC/FTE + gender split
12. `lib/kpi/evaluate.ts` — orchestrátor: jmeno KPI + period → EnrichedKPIEvaluation
13. Tests: `tests/kpi/catalog.test.ts`, `tests/analytics/*.test.ts`, `tests/kpi/calculators/*.test.ts`
14. Demo: `app/page.tsx` načte live KPI data a zobrazí jednoduché cards (no polish, bez designu — M3 je o UI)

### Out of scope (pro M3+)
- KPICard komponenta, sparkline visual, progress bar (UI = M3).
- Live Claude API (MockAIInsightProvider stačí).
- Calculators pro P2/P3 KPI (stub vrátí mock hodnotu).
