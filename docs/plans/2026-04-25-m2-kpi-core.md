# Implementační plán — Milník 2 (KPI core)

> **Stav k 2026-04-25 06:15 UTC:** M2 je implementovaný lokálně. Navazuje na M1 data layer a poskytuje první kompletní analytické API pro KPI karty.

## Cíl

Postavit deterministické KPI jádro pro prototyp:

1. Katalog všech 20 KPI z návrhu.
2. Výpočet hodnoty, statusu, trendu a sparkline.
3. Driver analysis po divizích.
4. Anomaly flagging nad 12M historií.
5. České rule-based narrativy.
6. Doporučená akce z katalogu.
7. Mock AI insight provider nad JSON obsahem.

## Hotové soubory

| Soubor | Účel |

| `lib/analytics/driver-analyzer.ts` | top divizní přispěvatelé změny |
| `lib/analytics/anomaly-detector.ts` | z-score anomaly flag |

## Výstup pro UI

Hlavní entrypoint:

```ts
const model = await buildKpiCardModel(
  mockDataProvider,
  'FLUCT',
  { period: { from: '2025-01-01', to: '2025-12-31' } },
  mockAIInsightProvider,
);
```

Vrací:

- `evaluation` — hodnota, status, trend, sparkline, data quality
- `drivers` — top přispěvatelé podle divize
- `anomaly` — z-score flag
- `narrativeCs` — český rule-based komentář
- `action` — doporučená akce
- `aiInsight` — mock AI komentář
